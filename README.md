# Sobr — архив выжимок созвонов

Локальный сайт для красивого хранения выжимок созвонов.

## Как открыть

Открой файл двойным кликом:

`C:\Users\User\my-project\Sobr\index.html`

## Что внутри

- `index.html` — главная страница сайта.
- `assets/style.css` — оформление.
- `assets/app.js` — логика фильтров, карточек и темы.
- `data/meetings.json` — данные в JSON.
- `data/meetings.js` — те же данные для открытия сайта напрямую через file://.

## Как добавлять новые созвоны

Добавлять новые объекты в `data/meetings.json`, затем обновлять `data/meetings.js`.
